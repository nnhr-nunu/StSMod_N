# キャラクターUIアイコン（コードから直接参照）

選択ボタン・マップマーカー・エネルギーアイコンなど。
ファイル名は C#（HypnosisCreator.cs / *Pool.cs）と一致させる必要があります。

## 配置するファイル

| ファイル名（固定）              | 用途                         | サイズ目安   |
|--------------------------------|------------------------------|--------------|
| char_select_char_name.png      | 選択画面のボタン（解禁時）   | 正方形〜縦長 |
| char_select_char_name_locked.png | 選択画面のボタン（ロック） | 同上         |
| character_icon_char_name.png   | 左上・統計などの小アイコン   | 64〜128px 角 |
| map_marker_char_name.png       | マップ上のマーカー           | 64〜128px 角 |
| big_energy.png                 | エネルギー大（カード左上）   | 74×74        |
| text_energy.png                | エネルギー小（説明文中）     | 24×24        |

※ コードが参照するのは上記の固定名。素材元は `*_nnhr.png` として残してよい。

## 仕様

- 拡張子: .png（透過推奨。小アイコン／マップは円形マスク＋透過）
- 命名: 固定名を変更しないこと（コード側と連動）
- サイズ目安（現状）:
  - character_icon / map_marker … 128×128（円形）
  - char_select / locked … 132×195（縦長）
- 差し替え後は dotnet publish が必要
