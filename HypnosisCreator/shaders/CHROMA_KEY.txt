# 見た目ライブ調整（Mod設定）

ゲーム内:
  Settings → Mod Settings → Hypno Creator

スライダーを動かすとその場の立ち絵／選択背景／カード絵へ反映される。
設定画面を閉じると自動保存（＝確定）。

## SelectBackground（キャラ選択の1枚絵）

| 項目 | 意味 |
|------|------|
| SelectBgOffsetX | 左右（px） |
| SelectBgOffsetY | 上下（px）。**プラス＝画面下へ**（顔が見切れるときは＋にする） |
| SelectBgZoom | 拡大 |

※ キャラ選択画面を開いた状態で調整すること（画面に背景ノードがないと効かない）

顔が上で見切れる例:
  SelectBgOffsetY を +40〜+120 くらいから試す

## CardArt（カード個別切り抜き）

1. CardTargetId にカードIDを入れる（例: `cooking_hypnosis`）
2. Offset / Zoom を調整（そのカードだけに保存される）
3. `_default` は「個別設定のないカード」向けの共通値
4. 「ListCardIds」ボタンで使えるID一覧がログに出る

## ChromaKey（立ち絵GB透過）

| 項目 | 意味 |
|------|------|
| ChromaSimilarity | 抜く範囲（緑が残る→上げる／靴などが欠ける→下げる）目安 0.16〜0.20 |
| ChromaSmoothness | 境界ぼかし |
| ChromaSpill | 緑かぶり抑制（白い靴が緑っぽいとき下げめ） |
| ChromaKeyR/G/B | 抜く色 |

ウォーターマークは画像を改変せず、シェーダーで角抜き＋足元の白いロゴ文字検出で消します
（立ち絵は Flip H ON 前提）。靴を守るため、明るい画素は抜けにくくしています。

※ 以前の Similarity 0.22 などが設定に残っていると靴が欠けます。
  そのときは ChromaSimilarity を 0.18 前後に戻してください。

## 反映の注意

- 初回導入後はゲーム再起動
- ゲーム起動中は mods への上書きがロックされることがある
- 画像差し替え後は .pck 再生成が必要
