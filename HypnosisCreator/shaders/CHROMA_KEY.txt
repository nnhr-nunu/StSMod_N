# 見た目ライブ調整（Mod設定）

ゲーム内:
  Settings → Mod Settings → Hypno Creator

スライダーを動かすとその場の立ち絵／選択背景／カード絵へ反映される。
設定画面を閉じると自動保存（＝確定）。

## セクション

### ChromaKey（立ち絵のグリーンバック透過）
| 項目 | 意味 |
|------|------|
| ChromaSimilarity | 抜く範囲（残る→上げる／欠けすぎ→下げる） |
| ChromaSmoothness | 境界ぼかし |
| ChromaSpill | 緑かぶり抑制 |
| ChromaKeyR/G/B | 抜く色（idle.png のGB色に合わせる） |

初期値は idle.png 実測色（約 RGB 8,161,53）向け。

### SelectBackground（キャラ選択の1枚絵）
| 項目 | 意味 |
|------|------|
| SelectBgOffsetX/Y | パン（左右／上下） |
| SelectBgZoom | 拡大（大きいほど寄る＝トリミング感） |

### CardArt（カード絵・全 Hypno Creator カード共通）
| 項目 | 意味 |
|------|------|
| CardOffsetX/Y | パン |
| CardZoom | 拡大 |

手札やカード報酬を出しながらスライダーを動かすと反映される。

## ファイル
- chroma_key.gdshader / chroma_key_material.tres … 立ち絵
- image_crop.gdshader / image_crop_material.tres … 選択背景・カード

## 反映の注意
- 初回導入後はゲーム再起動
- ゲーム起動中は mods フォルダへの上書きがロックされることがある
