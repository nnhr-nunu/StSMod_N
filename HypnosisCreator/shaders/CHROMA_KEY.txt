# 見た目ライブ調整（Mod設定）

ゲーム内:
  Settings → Mod Settings → Hypno Creator

スライダーを動かすとその場の立ち絵／選択背景／カード絵へ反映される。
設定画面を閉じると自動保存（＝確定）。

## SelectBackground（キャラ選択の1枚絵）

| 項目 | 意味 |
|------|------|
| SelectBgOffsetX | 左右（px） |
| SelectBgOffsetY | 上下（px）。**プラス＝画面下へ**（顔が見切れるときは＋にする） |
| SelectBgZoom | 拡大 |

※ キャラ選択画面を開いた状態で調整すること（画面に背景ノードがないと効かない）
※ 位置はシェーダーで画像をパンする。スライダーを動かすとその場で反映される

顔が上で見切れる例:
  SelectBgOffsetY を +40〜+120 くらいから試す

## CardArt（カード個別切り抜き）

1. CardTargetId にカードIDを入れる（例: `cooking_hypnosis`）
2. Offset / Zoom を調整（そのカードだけに保存される）
3. `_default` は「個別設定のないカード」向けの共通値
4. 「ListCardIds」ボタンで使えるID一覧がログに出る

## ChromaKey（立ち絵GB透過）

| 項目 | 意味 |
|------|------|
| ChromaSimilarity | 抜く範囲（緑が残る→上げる／靴などが欠ける→下げる）目安 0.16〜0.20 |
| ChromaSmoothness | 境界ぼかし |
| ChromaSpill | 緑かぶり抑制（白い靴が緑っぽいとき下げめ） |
| ChromaKeyR/G/B | 抜く色 |

## Watermark（右下ロゴ隠し）

※ **戦闘中**（idle 等の立ち絵が出ている状態）で調整すること。設定変更は即反映される。

| 項目 | 意味 |
|------|------|
| WatermarkCropBottom | 下端から何割を角抜きするか（目安 0.055〜0.09） |
| WatermarkCropSide | ロゴ側の横幅（目安 0.38〜0.55）。上げすぎると靴が欠ける |
| WatermarkLogoStrength | 白いロゴ文字検出の強さ（1＝既定。残るとき 1.2〜1.5） |

ロゴは素材 PNG の**右下**（flip_h 表示で画面左下）にある前提。旧版は UV 左を隠していて効かなかった。

PNG は改変せず、シェーダーで角抜き＋足元の白いロゴ文字検出で消します
（立ち絵は Flip H ON 前提）。靴を守るため、明るい画素は抜けにくくしています。

敗北時は同じシェーダーの dissolve（ノイズ消滅）が走ります（die 連番素材不要）。

パワー詠唱の白い粒子が青っぽく見える場合:
  素材の粒子に緑かぶりがあり、Spill で G だけ落ちて青が残るのが原因。
  シェーダーの sparkle_whiten（既定 0.9）で明るい粒子を白へ戻しています。

※ 以前の Similarity 0.22 などが設定に残っていると靴が欠けます。
  そのときは ChromaSimilarity を 0.18 前後に戻してください。

## 反映の注意

- 初回導入後はゲーム再起動
- ゲーム起動中は mods への上書きがロックされることがある
- 画像差し替え後は .pck 再生成が必要
