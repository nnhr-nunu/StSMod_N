最終確定: 2026-07-19（CSV更新反映）。実装・レビュー時はここを正とする。変更はユーザー指示があるときのみ。
カード一覧の正本: ユーザー配布の `ヒプノクリエイター.csv`（No.1〜104）。CSVに無いカードは実装しない。

## 廃止

- Dom/Sub 一致システムは廃止。性癖ギミックへ統合済み。

## カウント

- キーワード付きカード共通: **保留 / ターン開始時コスト−1 / 解決後コスト0のときだけプレイ可 / プレイ時廃棄**
- 「催眠系カウント」＝カウント（同一）
- トランスで非0プレイしても **廃棄する**
- **サポートカード**（アブノーマル・指折り数えて・練達・ゼロへの近道など、敵を取らない／自分・手札操作）はカウントキーワード無し
- **ポリネシアン催眠**はカウントキーワード付き（コスト6・パワー）

### トランスによるカウント制限解除

| 対象タイプ | 非0コストでもプレイ可になる条件 |
|------------|----------------------------------|
| 単体敵 | **その敵**がトランス状態 |
| 全体敵 | **敵の誰か**がトランス状態 |
| 自分／対象なし | 解除なし（サポートはキーワード無し） |

## トランス（敵デバフ）

- 「トランスN」＝**付与量 N**（1回の付与）。「トランス1を3回」＝付与を3回
- 条件「トランス3以上」＝**現在スタック ≥ 3**
- 減少: **プレイヤーターン開始**（Player side の SideTurnStart）に1
- **トランス性癖の刺さり**: 対象がトランス性癖を持つとき、**トランスが付与されるたび**に刺さり判定（付与1回につき1回）。連続トランスは3回付与なので最大3回刺さる。付与前にトランス性癖を目覚めさせるカードは、目覚め→付与の順

## 性癖

- 種類: SM / DomSub / アブノーマル / トランス（最大4。同種重複なし）
- 目覚め: スロット増加＋付与。同種は増えない
- **刺さり（カードタグ）**: 性癖タグ付きカードをプレイし、対象が同種を持つ（または必ず刺さる）とき
  - 複数ヒット攻撃でもプレイあたりタグ解決はまとめて（リプレイなら再判定）
  - 複数タグは**種類ごとに個別判定**（足蹴・壱佰捌煩悩・教祖化など）
  - 「必ず刺さる」でタグが複数でも、感度3000倍は**1回の刺さり**（必ず1回発動）。複数タグ個別が明記されているカードは個別
  - 破滅量: `max(8, ceil(MaxHP × 5%) + 7)`
- **教祖化**: すべての性癖カードが刺さる。**トランス性癖は含まれない**。複数タグは個別判定
- 括弧書きで破滅Nをカード末尾に表記。刺さり可能時はカードを黄色ハイライト（演出）

## 破滅・沼

- 破滅 = 本家 `DoomPower`
- 沼: 受ける破滅 **×1.5（乗算1回）**。ターン終了時に1減少

## リプレイ・本家要素

- リプレイ = `BaseReplayCount`
- スタン / 睡眠(`AsleepPower`) / 粘液(`Slimed`) / 本能(`Instinct`) / 弱体・脱力・毒・スロー・縮小・アーティファクト 等は本家準拠
- コマンド `Don't Move!`（表記はこの通り）

## 心臓レリック（CSV No.111〜）

- **希少な心臓**: 使用で効果。冒険中1回。使用済みはグレー（`IsUsedUp`）。No.86で再使用可能
- **非希少**: 入手時に最大HP／ゴールド等
- No.85 自己暗示: ランダム3つ（所持済み・戦闘中敵の心臓は除外）。戦闘開始トリガーを満たす
- No.86 心臓への渇望: 所持心臓をこの戦闘中再使用可能に。UG後は戦闘外でも再使用可
- No.68 解剖: 14＋心臓数×4（UG×7）、リーサルで心臓、廃棄
- 基底: `EnemyHeartRelic` / 数え上げ: `HeartInventory`
