最終確定: 2026-07-20（UG説明は差分のみ緑表示・全文差し替え禁止）。実装・レビュー時はここを正とする。変更はユーザー指示があるときのみ。

## 廃止

- Dom/Sub 一致システムは廃止。性癖ギミックへ統合済み。

## カウント

- キーワード付きカード共通: **保留 / ターン開始時（手札ドロー前）コスト−1 / コスト0のときだけプレイ可 / プレイ時廃棄**
- プレイヤー向けツールチップ: 「保留。ターン開始時にコストが1減少する。コストが0の場合にプレイできる。」（廃棄・トランス例外はキーワード文面には書かない）
- コスト−1は**ドロー前の手札**にだけかかる。今ターン引いたカードは元コストのまま（開幕の3コストも3のまま）
- 「催眠系カウント」＝カウント（同一）
- トランスで非0プレイしても **廃棄する**
- **サポートカード**（アブノーマル・指折り数えて・練達・ゼロへの近道など、敵を取らない／自分・手札操作）はカウントキーワード無し
- **ポリネシアン催眠**はカウントキーワード付き（コスト6・パワー）

### トランスによるカウント制限解除

| 対象タイプ | 非0コストでもプレイ可になる条件 |
|------------|----------------------------------|
| 単体敵 | **その敵**がトランス状態 |
| 全体敵 | **敵の誰か**がトランス状態 |
| 自分／対象なし | 解除なし（サポートはキーワード無し） |

## トランス（敵デバフ）

- 「トランスN」＝**付与量 N**（1回の付与）。「トランス1を3回」＝付与を3回
- 条件「トランス3以上」＝**現在スタック ≥ 3**
- 減少: **プレイヤーターン開始**（Player side の SideTurnStart）に1
- プレイヤー向けツールチップ: 「コストが0ではない状態でもカウントカードをプレイできるようになる」
- トランスを付与するカードは、ホバーにトランス説明を出す（`CanonicalVars` の `Trance` で自動付与）
- **トランス性癖の刺さり**: 対象がトランス性癖を持つとき、**トランスが付与されるたび**に刺さり判定（付与1回につき1回）
- **連続トランス（No.9）**: 1プレイでトランス1→沼1→トランス性癖目覚め、リプレイ2（計3回）。付与時点で性癖が無ければその回は刺さらない（未所持なら最大2回、所持済みなら最大3回刺さる）。沼はトランスのあと（トランスで沼が深まる演出）
- 付与前にトランス性癖を目覚めさせる他カードは、目覚め→付与の順

## カード説明・UG表記

- 説明文の**全文差し替え禁止**（`[gold]`キーワード色や性癖プレフィックスが消える）
- UGの定性差分は文中置換または末尾行追加で、追加／変更箇所だけ `[green]…[/green]`（`UpgradeDescriptionHooks`）
- 数値UGは `{Var:diff()}` を使う
- コスト−1／保留／廃棄消し／天賦のみのUGは枠UI側の表示に任せ、説明追記は必須ではない

## 性癖

- 種類: SM / DomSub / アブノーマル / トランス（最大4。同種重複なし）
- 目覚め: スロット増加＋付与。同種は増えない
- **同一プレイで目覚めた性癖は、そのプレイでは刺さらない**（次のカード／リプレイから有効）。必中（感度3000倍・壱佰捌煩悩など）は従来どおり
- **刺さり（カードタグ）**: 性癖タグ付きカードをプレイし、対象が同種を持つ（または必ず刺さる）とき
  - 複数ヒット攻撃でもプレイあたりタグ解決はまとめて（リプレイなら再判定）
  - 複数タグは**種類ごとに個別判定**（足蹴・壱佰捌煩悩・教祖化など）
  - 「必ず刺さる」でタグが複数でも、感度3000倍は**1回の刺さり**（必ず1回発動）。複数タグ個別が明記されているカードは個別
  - 破滅量: `max(8, ceil(MaxHP × 5%) + 7)`（沼があるとき付与時に×1.5）
- **敵の性癖ツールチップ**: 「○が性癖。性癖に刺さる行動を受けた時、破滅Nを得る。」（Nはその敵の実計算値。バフ行の性癖パワー／下部HUDの両方）
- **教祖化**: すべての性癖カードが刺さる。**トランス性癖は含まれない**。複数タグは個別判定
- 括弧書きで破滅Nをカード末尾に表記。刺さり可能時はカードを黄色ハイライト（演出）

## 破滅・沼

- 破滅 = 本家 `DoomPower`
- 沼: 受ける破滅 **×1.5（乗算1回）**。ターン終了時に1減少

## リプレイ・本家要素

- リプレイ = `BaseReplayCount`
- スタン / 睡眠(`AsleepPower`) / 粘液(`Slimed`) / 本能(`Instinct`) / 弱体・脱力・毒・スロー・縮小・アーティファクト 等は本家準拠
- コマンド `Don't Move!`（表記はこの通り）

## 心臓レリック（CSV No.111〜187）

- **すべて希少な心臓**: 右クリックで効果。冒険中1回。使用済みはグレー（`IsUsedUp`）。No.86で再使用可能（UG後は永続）
- No.85 自己暗示: 希少な心臓からランダム3つ（所持済み・戦闘中敵の心臓は除外）。戦闘開始トリガー相当で即発動（この戦闘のみの一時所持）
- No.86 心臓への渇望: 所持心臓をこの戦闘中再使用可能に。UG後は戦闘外でも再使用可
- No.68 解剖: 14＋心臓数×4（UG×7）、リーサルで心臓、廃棄
- 基底: `EnemyHeartRelic` / 数え上げ: `HeartInventory`
- **敵固有心臓の入手UX**: リーサル／寄生／心停止＋はすべて報酬画面の追加 `RelicReward`（`HeartCapture.TryAddExtraRelicReward`）。即時 Obtain は部屋が無いときのフォールバックのみ
- 例外: 自己暗示（この戦闘中の一時所持）・鼓動の共有（味方への近似共有）は従来どおり即時 Obtain
- カイザークラブは左右別心臓（`CrusherHeart` / `RocketHeart`、効果は同）
- 万足ムカデ／アックスマシン等の「バフを得る」心臓は、重ねがけ（No.86再使用含む）で回数増加

## カード個別メモ（CSV備考の要点）

- 完全掌握: DomSub。肩代わり。廃棄（UGで消滅）
- ぜーろっ: 攻撃値0（UGでブロックも0）
- 心停止催眠: カウント。3ターン後に心臓停止（ボスは2倍ターン）。トランス1。UGで停止時に追加レリック報酬。心停止デバフは残りターン表示。0になった後の対象行動終了時に即死。重ねがけで残りターン−1
- 連続指パッチン: 全敵に1ダメージ×5をX回。合計ダメージは戦闘中のみ表示（筋力・弱体反映）。廃棄（UGで保留）

- 調和: 脆弱等のデバフ影響なし。連撃は合計
- ミラーリング: 筋力非反映。敵の弱体は有効。廃棄（UGで消滅）
- 認知シャッフル: 性癖タグなし。悪魔化等（虚無化の強制終了なし）
- 侵食: アタックプレイのたびランダムな催眠カウントを手札へ（UGで性癖優先）
- 囁き: **山札**から選ぶ。廃棄
- Good!: コマンド追加（No.66）
- 暗示解除: 基本はエナジーのみ。UGでドローも
- 不動明王: デバフ解除＋アーティファクト＋付与してきた相手にダメージ
- マルチ用（ASMR・鼓動の共有・性癖パーリナイ・フリーハグ）: v1必須
- 上記4枚は `CardMultiplayerConstraint.MultiplayerOnly`（ソロのカード報酬・巻物箱・ショップ抽選から除外。マルチ時のみ出る）
