# Hypno Creator — 確定メカニクス（再質問禁止用）

最終確定: 2026-07-18。実装・レビュー時はここを正とする。変更はユーザー指示があるときのみ。

## 廃止

- Dom/Sub 一致システム（`DominationPower` / `SubmissionPower` / `TranceMatch`）は廃止。性癖ギミックへ統合済み。

## カウント

- キーワード付きカード共通: **保留 / ターン開始時コスト−1 / 解決後コスト0のときだけプレイ可 / プレイ時廃棄**
- 「催眠系カウント」＝カウント（同一）。呼び方はカウントで統一してよい
- トランスで非0プレイしても **廃棄する**
- **サポートカード**（アブノーマル・指折り数えて・練達・ゼロへの近道など、敵を取らない／自分・手札操作）はカウントキーワード無し。トランスの有無に関係なく通常コストで使える（2026-07-18 追加確定）

### トランスによるカウント制限解除

| 対象タイプ | 非0コストでもプレイ可になる条件 |
|------------|----------------------------------|
| 単体敵 | **その敵**がトランス状態 |
| 全体敵 | **敵の誰か**がトランス状態 |
| 自分／対象なし | 解除なし（サポートはキーワード無し） |

手札からのドラッグ開始（`CanPlay`）: 単体でも「誰かがトランス」なら掴める。着弾（`CanPlayTargeting`）は対象がその敵でトランス必須。

## トランス（敵デバフ）

- 敵へのスタック。「トランスN」表記＝**付与量 N**
- **完全掌握**などの条件「トランス3」＝**現在スタック ≥ 3**
- 減少タイミング: **プレイヤーターン開始**に1（ユーザー確定。弱体の本家タイミングとは別にこの仕様）
- カウント制限解除は上記表どおり

## 性癖

- 種類: SM / DomSub / アブノーマル / トランス（最大4。同種重複なし）
- 目覚め: スロットが増えてその性癖が付く。既にある種類は増えない
- **刺さり**: 性癖タグ付きカードをプレイし、対象が同種性癖を持つ（または「必ず刺さる」）とき
  - 1プレイ1回（複数ヒットでも1回）。**リプレイなら再判定**
  - 破滅量: `max(11, ceil(MaxHP × 5%) + 10)`
- 「必ず刺さる」: スロット無視で刺さり発動
- 「すべての性癖に目覚めさせる」系はトランス性癖も含む

## 破滅・沼

- 破滅 = 本家 `DoomPower`（ネクロバインダーと同仕様）
- 沼 = 独自デバフ。付与中は受ける破滅が **×1.5（乗算は1回。スタック数で倍率は増えない）**。ターン終了時に1減少

## リプレイ・同名本家要素

- リプレイ = 本家 `BaseReplayCount` 系（Transfigure 等と同じ）
- スタン / 睡眠(`AsleepPower`) / 粘液(`Slimed`) / 本能(`Instinct`) / 弱体・脱力・毒・スロー・縮小 等は **本家準拠**
- スライム催眠: 見た目をランダムなスライム敵に変更＋行動を粘液付与（`Slimed`）枚数上書き

## カード個別で確定済み（抜粋）

- ぜーろっ: 「1ターン3回」は無し（廃棄と矛盾のため削除）
- Crowl! → **Crawl!**
- 薬物催眠: カウント扱い（本文にカウント表記忘れ）
- 完全掌握: 単体。敵の攻撃→自分、自分の攻撃→自分
- 不意打ち催眠: 対象なしカードも自動プレイする
- ラポール: 「誰も攻撃していない」は全体判定
- 好き好き催眠: 敵のバフ行動の対象をプレイヤーに変更
- 認知シャッフル: 悪魔化／大蛇化／虚無化（強制終了なし）／死神化／反響化。生成カードはエセリアル＋廃棄
- ミラーリング: アタック意図があるときのみ。筋力等は反映せず意図数値どおり
- 初心者向け催眠: 次にプレイする性癖カードの **該当タグ全部** を植え付け
- 脳くちゅ催眠: 移し替え先に味方は含まない
- 植物寄生催眠: キル不要。戦闘終了で心臓。いつ死んでも可
- 寝かしつけ: 本家睡眠＋zzz。起床後は元の予定行動
- お仕置き: その敵がプレイヤーを攻撃した回数のみ
- 練達: 戦闘終了後に **永続** UG
- トランスに溶けゆく: トランス **付与回数** 累計
- 侵食: コピーも対象
- 壱佰捌煩悩: コスト上昇は戦闘中永続。刺さり最大4回
- 暗示解除: トランス全解除。スタック数＝エナジー＆ドロー
- ご主人様の言うとおり / 性倒錯への理解: **プレイ時**（刺さり時ではない）。77備考の「種類カウント」は消し忘れ→無視
- 受容の需要: 被ダメ **ヒット数**
- アンコール: 廃棄を取り消して手札へ戻す
- 剥き出しの本能: 本家 `Instinct` と同じ
- マルチ用カード（ASMR・鼓動の共有・性癖パーリナイ・フリーハグ）: **v1必須**
