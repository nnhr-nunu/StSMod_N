### 画像ファイル（サムネ）

クラス名から自動でファイル名が決まります。  
**内部ID**は `COOKING_HYPNOSIS` のような形式（こちらで命名案を出します）。

| ファイル | サイズ目安 | ファイル名（例） |
|----------|------------|------------------|
| 手札・一覧用 | 500×380 または 250×190 | `cooking_hypnosis.png` |
| 拡大鑑賞用 | 1000×760 または 250×350 | `big/cooking_hypnosis.png` |
| β版差分（任意） | 同上 | `beta/cooking_hypnosis.png` |

**配置先（渡すときのフォルダ構成）:**

```
cards/
  cooking_hypnosis/
    cooking_hypnosis.png      ← 手札用
    big/
      cooking_hypnosis.png    ← 拡大用
    beta/                     ← 任意
      cooking_hypnosis.png
```

または ZIP 内で `カード名（日本語）/画像ファイル` でも可。  
ファイル名が分からなければ、画像だけ渡してカード名をフォルダ名にしてください。

## 付録 B: 完全掌握（将来メカニクス）

トランス軸の到達点として検討中:

- 条件: Dom と Sub が一定以上一致、または専用カードで「完全掌握」付与
- 効果: 敵 HP を 1 にし、睡眠（またはスタン）で行動不能を維持
- 結果: 任意の小ダメージでリーサル → 心臓取得が確実に成立

該当カードがある場合、上の「完全掌握」欄に `トリガー` / `持続` / `解除条件` を書いてください。

## 付録 C: 最低限これだけあれば実装可能

3. 種類・レア度・コスト
4. 効果の文章（数値込み）
5. サムネ画像（手札用1枚でも可）
